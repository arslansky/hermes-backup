# Archive

This directory is reserved for old backups, historical snapshots, and files that are no longer actively used by the live VMs.

## What goes here
- Old cron output dumps
- State database backups
- Historical agent session exports
- Deprecated scripts that are kept for reference only

## What does NOT go here
- Active scripts — those live in `../scripts/`
- Active skills — those live in `../skills/`
- Setup documentation — those live in `../setup-guide/`
